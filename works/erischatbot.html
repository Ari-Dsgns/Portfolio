ERIS Chatbot

University FAQ assistant built with React, Vite & Tailwind CSS

View source code → GitHub Live Demo

Summary

ERIS is a chatbot designed to answer frequently asked questions from university students. Inspired by the solar system, it combines modern UI, accessibility, and attractive animations to engage young, tech-savvy users.

Project goal:

Build an accessible FAQ chatbot with a playful, futuristic visual identity while applying advanced React skills and Tailwind CSS for rapid prototyping.

Technologies used:

React.js

Vite

Tailwind CSS

JavaScript ES6+

My role:

I designed and developed the entire project:

Created brand identity and UI kit in Figma

Built a responsive chat interface in React & Tailwind

Developed interactive logic and bot responses in JS

Implemented accessibility standards (WCAG 2.1 AA)

Deployed via Vercel

Challenges and solutions:

Ensuring keyboard navigation, screen reader compatibility, and fluid conversation flow while maintaining the brand's solar-system inspired animations. Solved using semantic HTML, ARIA roles, and Tailwind's utility-first approach for rapid design iterations.

What I learned:

• Building accessible interactive components with Tailwind CSS

• Combining React with Vite for fast dev workflow

• Designing chatbot UI for clarity and engagement

• Structuring scalable JS logic for FAQs
