JUSTGOODSTUFF | E-COMMERCE | REACT JS | CSS

This is an e-commerce site for purchasing various types of products. This time, I used React JS for the app's structure and logic, fetching products from Firebase and applying styles using CSS. You can browse the app, filter products by category, add them to the cart, and finally complete the purchase to obtain a purchase ID. The result is a functional, fast, and attractive website.
